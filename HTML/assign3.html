Unordered List (Default - Discs)

  • Item A

  • Item C

This is an unordered list with circle bullet points:

  • Item 1

  • Item 3

Unordered List (Square)

This is an unordered list with square bullet points:

  • Item X
  • Item Y
  • Item Z

Unordered List (None)

This is an unordered list with no bullet points (using list-style-type: none;):

  • Item 1
  • Item 2
  • Item 3

  1. First item
